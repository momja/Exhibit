# Remove Open action from artifact cards

The artifact card currently shows both 'Details' and 'Open' links. The 'Open' action should be removed altogether; the card itself (or the Details action) should be the only way to open/view an artifact.

## Acceptance Criteria

- The 'Open' link is removed from all artifact cards
- Clicking the artifact card opens the artifact in the viewer
- The 'Details' link navigates to the artifact metadata/detail page
- Behavior is consistent across all artifact cards

# Move Upload Artifact to its own page

The Library page currently mixes artifact upload functionality with artifact browsing. The 'Upload Artifact' form should be moved to its own dedicated page so that the Library is primarily for browsing, searching, and managing existing artifacts.

## Acceptance Criteria

- Upload Artifact has a dedicated route/page
- Library page shows only artifact browsing/search without the upload form
- Navigation provides a clear way to reach the Upload page
- Existing upload behavior is preserved

# Delete button in artifact editor

As a user, I should have the choice to delete an artifact completely, with a confirmation modal. This will permanently delete the artifact and all associated data in the localstorage shim and otherwise. Due to the irreversable nature of the action, users should receive an appropriate warning module with statement: Are you sure you want to delete this artifact? The action cannot be reversed and all data will be lost.
